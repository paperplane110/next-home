网易云外部播放器链接参数

```js title="player.js" {1,3}
<iframe 
    className="rounded-md transform"
    width={200} 
    height={430} 
    src="//music.163.com/outchain/player?type=0&id=5050009781&auto=1&height=66"
>
</iframe>
```

播放器目前探索出来两种类型：

- 歌单播放器
- 单曲播放器

播放器由两部分组成：首先是播放器容器，样式由 iframe 定义；其次是播放器本身，这部分样式没办法直接修改，需要通过修改 src 中的请求参数来实现

## iframe 参数 

- width: 播放器宽度
- height: 播放器高度

## src 参数

- type: 播放器类型，0 为歌单播放器，2 为歌单播放器单曲播放器
- id: 音乐/歌单 id
- auto: 是否自动播放，1 为自动播放，0 为不自动播放
- height: 播放器高度，决定了播放器的样式：大、中、小

| height | 样式 | 专辑封面 | 进度条 | 歌单列表 |
| --- | --- | --- | --- | --- |
| 非 66、32 像素 | 大（默认样式） | 有 | 有 | 有 |
| 66 | 中 | 有 | 有 | 无 |
| 32 | 小 | 无 | 有 | 无 |

- bg（可选）:背景颜色，16 进制颜色值，默认值为 #f5f5f5

若歌曲需要会员，则歌单播放会暂停，不会自动跳过
